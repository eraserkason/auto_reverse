---
name: logging-output
description: 将特定请求的响应保存到日志文件
---

将请求2和请求3的完整 HTTP 响应保存到独立的日志文件中。每个日志文件应包含响应状态行、响应头部和响应正文。
1.  对于请求2的响应，保存到文件 `log_request_2.json`。由于响应是 JSON 格式，可以原样保存，确保格式正确。
2.  对于请求3的响应，保存到文件 `log_request_3.json`。响应也是 JSON 格式，原样保存。

确保文件写入操作不会覆盖已有文件（除非明确要求），并考虑添加时间戳或序列号以区分多次运行的结果。日志文件用于后续分析和调试。
